Мощный Telegram-бот для автоматической категоризации видеофайлов с помощью Google Gemini AI. Бот анализирует названия видеофайлов, автоматически группирует их по темам, генерирует ключевые слова и предоставляет удобный поиск с элементами искусственного интеллекта.

## ✨ Возможности

### 🤖 AI-категоризация
- Автоматический анализ названий видеофайлов с помощью **Google Gemini 1.5 Flash**
- Интеллектуальная группировка по релевантным темам (Спорт, Кулинария, Обучение, Путешествия и др.)
- Генерация 2-4 ключевых слов для каждого видео на русском языке

### 🔍 Два режима поиска
- **Текстовый поиск** — поиск по названиям файлов и ключевым словам
- **Семантический поиск** — поиск по смысловой близости с использованием эмбеддингов Gemini

### 🧠 Семантический поиск на эмбеддингах
- Векторизация всех названий через `text-embedding-004`
- Поиск по косинусному сходству с запросом пользователя
- Кэширование эмбеддингов в pickle-файл для быстрой загрузки

### 💬 Интерактивные кнопки
После каждого поиска появляются кнопки для первого результата:
- **📂 Похожие по теме** — показать все видео из той же категории
- **🧠 Похожие по смыслу** — найти семантически близкие видео

### 📊 Статистика и управление
- Команда `/stats` показывает общую статистику базы
- Команда `/reload` перезагружает CSV без перезапуска бота
- Гибкая система ограничения доступа по Telegram ID

### 💾 Сохранение данных
- Результаты категоризации сохраняются в **CSV** файл
- Эмбеддинги кэшируются в **pickle** файл
- Поддержка инкрементального обновления

## 📋 Требования

### Системные требования
- **Python**: 3.8 или выше
- **ОС**: Windows, macOS, Linux
- **Интернет**: Стабильное подключение для API запросов
- **Память**: ~100MB + размер эмбеддингов (~1KB на видео)

### API ключи
1. **Google Gemini API** — [получить бесплатно](https://aistudio.google.com/)
2. **Telegram Bot Token** — [создать через @BotFather](https://t.me/BotFather)

### Поддерживаемые форматы видео
`.mp4`, `.avi`, `.mkv`, `.mov`, `.wmv`, `.flv`, `.webm`, `.m4v`, `.mpg`, `.mpeg`

❓ Часто задаваемые вопросы
🤔 Как добавить новые видео в существующую базу?

    Поместите новые видео в ту же папку

    Перезапустите скрипт и выберите режим 1 (обработать видео)

    Затем выберите режим 4 (обновить эмбеддинги)

    Запустите бота режим 2

💰 Сколько это стоит?

Бесплатно! Google предоставляет щедрые бесплатные квоты:

    Gemini 1.5 Flash: 1500 запросов/день

    Embedding API: 1500 запросов/день

Для большинства пользователей этого более чем достаточно.
🔒 Безопасно ли отправлять названия файлов в Gemini?

Да. Google гарантирует, что данные, отправленные через API, не используются для обучения моделей и не сохраняются. Подробнее в политике конфиденциальности Google AI.

Да! Измените промпт в методе build_prompt() класса VideoProcessor:

📱 Как запустить бота на Android/iOS?

Бот работает на сервере. Вы можете:

    Запустить скрипт на своём компьютере (должен быть включен)

    Использовать облачный сервер (DigitalOcean, AWS, PythonAnywhere)

    Использовать Raspberry Pi как домашний сервер

🔄 Как часто обновлять эмбеддинги?

Только при добавлении новых видео. Существующие эмбеддинги не меняются.
💾 Сколько места занимают эмбеддинги?

Примерно 3-4 KB на одно видео. Для 10,000 видео — около 30-40 MB.
🐛 Устранение неполадок
Ошибка: ModuleNotFoundError: No module named 'google.generativeai'
bash

pip install --upgrade google-generativeai

Ошибка: telebot.apihelper.ApiException: Unauthorized

Проверьте правильность токена бота. Получите новый у @BotFather.
Ошибка: FileNotFoundError: Папка ... не существует

Укажите полный абсолютный путь к папке:

    Windows: FOLDER_PATH = "C:\\Users\\Username\\Videos"

    Linux/macOS: FOLDER_PATH = "/home/username/Videos"

Ошибка: google.api_core.exceptions.ResourceExhausted: 429

Превышен лимит запросов. Увеличьте REQUEST_DELAY в настройках:
python

REQUEST_DELAY = 5  # Было 2 секунды

Ошибка: JSONDecodeError от Gemini

Модель иногда возвращает некорректный JSON. Код автоматически делает повторные попытки. Если ошибка повторяется часто:

    Уменьшите BATCH_SIZE до 100 или 50

    Попробуйте модель gemini-1.5-pro

Бот не отвечает на команды

    Проверьте интернет-соединение

    Проверьте, не заблокирован ли бот в Telegram

    Посмотрите логи на наличие ошибок

    Перезапустите бота

Медленная работа семантического поиска

    Убедитесь, что эмбеддинги загружены (проверьте через /stats)

    При первом запросе эмбеддинги загружаются с диска — это нормально

    Последующие запросы будут мгновенными
